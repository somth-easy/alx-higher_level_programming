============================
How to Use 3-say_my_name.py
============================
This module defines a name-printing function 'say_my_name(first_name, last_name="")'.

Usage
=====

'say_my_name(...)' prints the provided first name and last name.

::

	>>> say_my_name = __import__('3-say_my_name').say_my_name
	>>> say_my_name("John", "Doe")
	My name is John Doe

::

	>>> say_my_name("Alice")
	My name is Alice 

Arguments
=========

The function expects two string arguments: 'first_name' and 'last_name' (optional).

::

	>>> say_my_name("John", "Doe")
	My name is John Doe

::

	>>>say_my_name("John", None)
	# Expected: Traceback (most recent call last):
	# TypeError: last_name must be a string

Non-Strings
===========

If either 'first_name' or 'last_name' is not a string, a TypeError is raised.

::

	>>> say_my_name(42, "Doe")
	Traceback (most recent call last):
	TypeError: first_name must be a string

::

	>>> say_my_name("John", 42)
	Traceback (most recent call last):
	TypeError: last_name must be a string

::

	>>> say_my_name(None, "Doe")
	Traceback (most recent call last):
	TypeError: first_name must be a string

::

	>>> say_my_name("John", None)
	Traceback (most recent call last):
	TypeError: last_name must be a string
